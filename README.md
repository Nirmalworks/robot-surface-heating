# Robot Surface Heating (UR10e + FLIR Lepton)

This is a study copy of a robotics surface-heating project that uses a
Universal Robots **UR10e** arm with multiple **FLIR Lepton 3.5** thermal
cameras, built on **ROS 2 Humble**. It was assembled by merging four
folder snapshots into one clean tree so the code can be read and
referenced.

> The original project's full documentation lives in
> [`robot_surface_heating_multi_flir/README.md`](robot_surface_heating_multi_flir/README.md)
> — start there for setup, dependencies, and run instructions.

## Layout

```
.
├── libuvc/                              # Third-party FLIR USB camera driver
│                                        # (BSD-licensed, see libuvc/LICENSE.txt)
└── robot_surface_heating_multi_flir/
    ├── README.md                        # Original project README — start here
    ├── robot_surface_heating/           # ROS 2 Humble workspace
    │   └── src/
    │       ├── thermal_camera/          # FLIR Lepton driver, calibration,
    │       │                            # CAD registration, point-cloud ops
    │       ├── thermal_camera_interfaces/# Custom ROS 2 msg/srv definitions
    │       ├── thermal_moveit_config/   # MoveIt 2 config for planning
    │       ├── ur_robot_driver/         # UR10e ROS 2 driver + launch files
    │       ├── Universal_Robots_ROS2_Description/  # URDF / meshes
    │       ├── dual_ur_ros_2/           # Dual-arm description
    │       ├── robot_interface/         # High-level robot motion API
    │       ├── lerobot_interface/       # LeRobot integration
    │       ├── fake_gripper_publisher/  # Gripper state stub for sim
    │       └── Universal_Robots_ROS2_Tutorials/
    └── robot_surface_heating_old_ros1/  # Legacy ROS 1 reference code
                                         # (Motoman GP12 + heat gun)
```

## Notes on what's *not* here

To keep the repo under GitHub's size limits, the following were excluded
(see [`.gitignore`](.gitignore)):

| Excluded | Why |
|---|---|
| `build/`, `install/`, `log/` | colcon build outputs — regenerated by `colcon build` |
| `.git/` from source snapshots | starting fresh history |
| `coldest_node_data/` (400 MB) | experiment result recordings (`.npz`) |
| `thermal_npz_datasets/` (99 MB) | saved thermal captures for offline use |
| `thermal_data_bag/` (18 MB) | ROS 2 bag recordings |
| `calibration_data/` (26 MB) | full calibration captures, tied to original hardware |
| `nd-composite-core.txt`, `nd-metal-curved.txt`, `data.rrd` | large point-cloud / Rerun viewer dumps |

A small representative sample of the calibration outputs is kept at
[`robot_surface_heating_multi_flir/robot_surface_heating/src/thermal_camera/calibration_data_samples/`](robot_surface_heating_multi_flir/robot_surface_heating/src/thermal_camera/calibration_data_samples)
for reference — see the README in that folder for what each file is.

## Building

From `robot_surface_heating_multi_flir/robot_surface_heating/`:

```bash
colcon build
source install/setup.bash
```

Refer to the project README for camera setup, udev rules for the FLIR
Lepton, calibration procedures, and launch commands.
